# Règles d'implémentation pour le projet Boutique OS

- **NE JAMAIS TOUCHER AU FRONTEND** : Interdiction absolue de modifier les fichiers de l'interface utilisateur frontend (fichiers React, TSX, JS, CSS, HTML, PWA, etc.). Toutes les tâches doivent être résolues et implémentées exclusivement côté backend (scripts SQL, migrations Supabase, configuration de base de données).
